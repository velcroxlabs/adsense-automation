# AdSense Automation Project

## Objetivo
Crear un ecosistema de "Nicho Automático" para monetizar con Google AdSense con filosofía Zero-Human-Touch.

## Estructura
- `/keywords` - Investigación de keywords y nichos
- `/scrapers` - Scripts de scraping de datos
- `/content` - Generación de artículos con LLMs
- `/website` - Código del sitio web (Astro)
- `/automation` - Scripts de automatización (GitHub Actions, Make)
- `/monitoring` - Dashboards y métricas
